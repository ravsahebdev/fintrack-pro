# 💰 FinTrack Pro

A modern and feature-rich Personal Finance Tracker built using HTML, CSS, and Vanilla JavaScript. FinTrack Pro helps users manage their income, expenses, and financial activities with a clean dashboard, interactive analytics, and secure local data storage.

---

## ✨ Features

- 🔐 User Authentication (Login & Register)
- 💰 Income & Expense Management
- 📊 Financial Dashboard
- 📈 Interactive Cash Flow Chart
- 🔍 Search & Filter Transactions
- 👤 User Profile Management
- 🌍 Multi-Currency Support
- 🌙 Dark / Light Mode
- 💾 LocalStorage Data Persistence
- 📱 Fully Responsive Design

---

## 🚀 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- Chart.js
- LocalStorage API
